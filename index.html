Compression Techniques

JPEG

Lossy compression removing fine details. Lower quality = smaller file with more artifacts.

WebP

Modern format with superior compression ratios. Supports both lossy and lossless modes.

AVIF

Next-gen format. Better than WebP with smaller file sizes and wider color gamut support.

Progressive JPEG

Multi-pass encoding that loads gradually from low to high quality, improving perceived load speed.

Chroma Subsampling

4:2:0 reduces color info while preserving luminance, exploiting human visual perception.

Color Quantization

Reduces the number of distinct colors, decreasing file size while maintaining visual appearance.

Grayscale

Converts to grayscale using luminance formula, reducing color data by 66%.

Floyd-Steinberg Dithering

Error diffusion creates the illusion of color depth with a limited palette via spatial dithering.

Run-Length Encoding

Compresses consecutive identical pixels into count-value pairs. Effective for uniform areas.

K-means Clustering

ML-based color reduction finding an optimal palette while preserving image quality.

⬡ Mesh Simplification

Reduces polygon count of 3D models (OBJ, STL, PLY) using QEM vertex clustering.

☁ Point Cloud Decimation

Voxel grid downsampling groups nearby points and keeps their centroid, removing redundant data.
